# BOOKING-BUS
🚌 Bus Ticket Booking System (C++)A lightweight, terminal-based Bus Seat Reservation System written in C++. This program allows users to view real-time seating charts, book single or group tickets(single entery at a time), automatically find the rows that have consecutive available seats, calculate age-based discounts, and handle mock payment processing.🌟 Key FeaturesInteractive Seating Chart: Displays an 8*5 seating layout ([0] for available, [B] for currently selected, and [X] for booked seats).Automatic Consecutive Seat Finder: For group bookings (up to 5 seats), the system scans the chart and identifies rows with enough adjacent free seats.Age-Based Discount System:Standard Fare: $5 per ticket.50% Discount: Senior citizens (age<=60 years) and minors (age< 18 years) pay only $2.50 per ticket.Dynamic Billing & Payment: Calculates final costs dynamically and verifies payment input prior to confirming the reservation.
Persistent Session Memory: Tracks booked seats in memory using static structures throughout the program's execution loop.
🛠️ Built WithLanguage: C++Standard Libraries: <iostream>, <string>
Steps to Book a Ticket:Select Option 2 to initiate booking.Enter the number of seats you wish to reserve.If booking multiple seats (up to 5), opt-in for the consecutive seat finder to see which rows can fit your group.Input the desired Row (1–8) and Column (1–5) numbers.Enter the age of each passenger to automatically calculate discounts.Enter the exact total amount to complete the payment and lock in your seats!
📋 Seating LegendSymbolDescription[0]Available: Seat is open for booking.[B]Selected: Seat is currently selected in your transaction.[X]Occupied: Seat has been successfully booked.
